# BACKEND_CHECKLIST

Практический backend-чеклист для проекта `MART` с учетом текущего стека:

- `Bun`
- `Elysia`
- `Prisma v7`
- `PostgreSQL`
- `prismabox`
- `@elysiajs/openapi`
- `Docker`

Документ учитывает текущее состояние проекта:

- базовый backend уже поднят;
- auth-инфраструктура уже заложена;
- ORM и база данных уже подключены;
- проект уже завернут в Docker.

Цель следующего этапа: не просто добавлять отдельные эндпоинты, а последовательно собрать доменную backend-архитектуру под Unity-клиент и музейный продукт.
